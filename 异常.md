> try-catch捕获异常

``` java
try{
    xxx
}catch(异常类名 变量名){
    
}
```

![image-20250528212918136](异常.assets/image-20250528212918136.png)

- 如果有多个异常，就用多个catch，或者在catch括号里用“|”

- 如果出现异常了，try后面的代码也不执行了

> ctrl+alt+t可以用xx语句块包含被选中的代码
